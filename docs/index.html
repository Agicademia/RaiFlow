Capabilities

Everything you need for AI compliance

From your first commit to production deployment, RaiFlow keeps you on the right side of the EU AI Act.

Zero-Friction Init

Scans your project for AI framework imports (LangChain, Ollama, OpenAI, Transformers...), infers your risk level, and writes a pre-filled raiflow.yaml and GitHub Actions workflow in seconds.

Static Compliance Gate

Eight checks (MVP) against your manifest covering Articles 9–14, risk assessment existence, disclosure flags, oversight endpoints, logging middleware, bias detection, and toxicity testing.

Semantic LLM Evaluation

Pass --enable-llm-checks to go beyond file existence, score your actual documentation quality against Annex IV criteria using a local or cloud LLM.

Live Dashboard

A real-time browser dashboard streams check results as they complete via SSE. Download JSON reports, explore per-article regulatory citations, and see your compliance score at a glance.

CI/CD Native

Four pipeline stages (pre-commit → ci → pre-deploy → post-deploy) with auto-generated GitHub Actions workflows. Blocks merges and deployments on failure. Zero dashboard in CI.

De Jure Engine

An iterative LLM-as-a-Judge pipeline that interprets legal text and extracts machine-readable compliance rules directly from regulatory documents.

Workflow

From zero to compliant in minutes

1

Install and initialise

Run pip install raiflow then raiflow init in your project root. RaiFlow scans your Python files, infers your EU AI Act risk level, and writes raiflow.yaml plus a GitHub Actions workflow.

2

Fill your manifest

The generated raiflow.yaml is pre-filled and commented. Add your risk assessment path, override endpoints, dataset path, and any other evidence your checks require.

3

Run checks

raiflow check --stage ci opens a live dashboard and streams results. Each check returns a score, threshold, status, and a precise remediation hint when it fails.

4

Push — CI does the rest

The auto-generated workflow runs on every pull request. Compliance failures block merges and deployments. A signed JSON report is attached as a build artifact for audit trail purposes.

Quick Start

Up and running in 60 seconds

terminal 
# 1. Install
pip install raiflow

# 2. Scaffold manifest + GitHub Actions workflow
raiflow init

# 3. Run compliance checks with live dashboard
raiflow check --stage ci

# 4. Headless mode for CI (auto-detected, but explicit is fine)
raiflow check --stage ci --no-dashboard

# 5. Semantic evaluation with an LLM (Ollama or API key)
raiflow check --stage ci --enable-llm-checks

All checks passed (8 checks, stage=ci)
